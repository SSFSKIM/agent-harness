# Horizons

Strategic proposals from the `scout` skill — the system's divergent agent. Each
horizon ranks **Tier 1** next-initiative bets (within the Mission + Core Axioms)
and flags **Tier 2** foundational challenges (compelling ideas that would require
the Mission or an axiom to evolve — escalated for a human re-decision, never
enacted). Decision-support a human reads; a chosen Tier-1 pick becomes a
product-spec via the `product-design` flow. The scout **proposes, never enacts**.
