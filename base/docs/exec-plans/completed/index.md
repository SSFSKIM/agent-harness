# Completed ExecPlans

Landed ExecPlans are archived here, moved from `../active/` by the `execplan`
completion gate once the gate is GREEN and the plan's review budget is satisfied
(see `docs/PLANS.md`). They are a historical record — terminal by design, not
re-opened.

`exec-plans/` is not an indexed category; the live picture is the derived
`nav.py roadmap`. This `index.md` is a lifecycle guide, not a listing, and
exists so the directory is self-describing.
